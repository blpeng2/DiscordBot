## ❗ Features 
 /+ `this`하면 사용가능합니다 !
* **🎵Music**: `곡추가` `곡삭제` `재생` `플레이리스트` `일시정지` `스킵`
* **🎮Game**:  <br>
↳     **/끝말잇기** : `생성` `방` `참가` `시작` `종료` <br>
↳     **/주식게임** : `주식생성` `주식현황` `구매` `판매` `지갑`
